v1.0.1
2025 年 1 月
最新
✨ 新增功能
  • 中英双语切换：完整支持简体中文 / English
  • 智能语言检测：默认跟随浏览器语言，非中文区自动切英文
  • 设置页新增语言选择器（跟随系统 / 中 / EN）
  • PWA Service Worker：支持完整离线访问
  • PWA screenshots：Chrome「添加到主屏」时展示 3 张应用截图
  • 应用文档矩阵：隐私政策 / 用户协议 / 技术支持 / 更新日志 / 欢迎页 全部上线
⚡ 优化
  • 引入 react-i18next 框架，支持按命名空间懒加载
  • 所有 UI 文案按命名空间拆分（common/calendar/schedule/settings/almanac/tools）
  • 新增 platform 工具函数，自动识别 Capacitor 原生壳与 Web 环境
  • Service Worker 智能跳过原生壳，避免 iOS/Android WebView 报错
🛠 修复
  • 修复 Capacitor 监听器在 React StrictMode 下重复注册问题
  • 修复 SwipeToDelete 在横向/纵向滑动方向锁定的边界条件
